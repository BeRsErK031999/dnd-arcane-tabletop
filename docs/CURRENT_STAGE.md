# Current Stage

## Текущий статус

Этапы 23-27 завершены и проверены.

## Результат

- Master и player canvas используют независимые сохранённые viewport.
- Компактный блок `Экран игрока` управляет отдельным окном через typed IPC: `ON/OFF`, `Fullscreen`, `Clear` и публикация `Активная сцена`.
- Выбор сцены в мастерской не меняет публичный экран. Публикация выполняется только явным действием.
- Изменения уже опубликованной сцены — карта, сетка, измерения, fog и видимые объекты — сразу переиздают player-safe projection.
- Player page в режиме сцены содержит только canvas и controls масштаба/центрирования; слои ГМ, master notes, приватные token state и метаданные не выводятся.
- Слои `Карта / ГМ / Токены` доступны с клавиатуры: стрелки, `Home`, `End` и roving focus.

## Проверка

- Unit tests включают регрессии для сохранения публичной сцены, live-проекции и очистки player screen.
- Browser smoke-проверка создала кампанию и сцену, опубликовала её и подтвердила отдельный player-view без лишней панели.
- Перед релизом выполнены `npm run lint`, `npm run typecheck`, `npm run test`, `npm run build` и `npm run dist:win`.
