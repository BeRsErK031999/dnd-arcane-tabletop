# Current Stage

## Текущий этап

Этап 22. Рабочая область по макету.

Статус: выполнено и проверено.

## Результат

Master UI перестроен вокруг постоянной центральной сцены: проектные операции остались в стартовой библиотеке, а готовые инструменты мастера доступны из компактного dock и открываются поверх canvas без изменения доменных контрактов или сохранения.

## Реализовано в этапе 22

- Desktop shell сокращён до 82 px и содержит компактную кнопку `Проекты` и пять инструментов редактора.
- На ширине до 840 px навигация становится горизонтальным верхним dock.
- Сцена остаётся смонтированной и визуально доминирует при переключении инструментов.
- `Ассеты`, `Бой`, `Заметки` и `Игроки` работают как overlay drawer; повторный выбор и `Escape` закрывают drawer.
- После keyboard-закрытия фокус и `aria-current` возвращаются на `Сцены`.
- Блок материалов сцены сворачивается, расширяя canvas.
- Создание сцен открывается по требованию, а переключение выполняется из компактной полосы.
- Дублирующий campaign CRUD удалён из workspace без изменения `useCampaignsStore`, JSON persistence, autosave и transfer contract.
- Кнопка возврата сохраняет проект; повторный запуск восстанавливает выбранный проект и активную сцену.

## Критерии готовности

- Central workspace проверен на 1366x768, 768x720 и 360x720.
- На tablet/mobile нет горизонтального переполнения страницы.
- Asset Manager доступен в полноширинном мобильном drawer.
- `Escape` закрывает drawer и возвращает keyboard focus на `Сцены`.
- Возврат в библиотеку и повторный запуск сохраняют активную сцену.
- Browser console не содержит errors или warnings.
- `npm run lint`, `npm run typecheck`, `npm run test`, `npm run build` и `npm run dist:win` проходят.
- Исходный Word и его временный lock-файл не входят в commit.

## Следующие риски

- Пользовательские слои нельзя отождествлять с техническими `map/grid/objects/tokens/master/fog`: fog и measurements должны остаться независимыми.

- Переключатель активного слоя не должен сбрасывать selection, viewport или undo/redo history.
- Новые токены должны использовать размер клетки по умолчанию без изменения существующих canvas objects.
